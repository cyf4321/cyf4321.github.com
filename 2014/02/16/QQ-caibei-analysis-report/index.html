腾讯 QQ 彩贝产品体验分析报告

Feb 16th, 2014 | Comments

这是一篇这个周末完成作业，整理删减后发到这里。另外，今天还发生一件大事，Cash 当爹了、成太当妈了，在这里也恭喜他们。

彩贝现状

从官网上可以看出，彩贝积分是与众多 B2C 合作的覆盖数码、食品、百货、服装、旅游、娱乐、信用卡等生活服务的通用积分返利平台。除了 B2C 商家的实物商品以外，还能兑换腾讯自家的服务。

特点是将现金返利变为彩贝积分，将返利模式变为福利模式，削弱了对商家品牌损伤的问题。

从用户层面来看，通过 QQ 号登录，建立能进行通兑的账户体系，成为通用积分消费平台，消耗手上用不掉的各个商家的积分。用户能用 QQ 号直接登录，无须注册，使登录更加便捷，避免多账号到账管理账号麻烦。

一直以来我都认为，本身消费返积分模式对大众消费者吸引不会太大。如果不返积分，用户依然会去消费和购物，这是刚性需求。但如果加上积分，就可以形成差异， 而且彩贝的优势是用户获得的是通用积分，积分并不限于只在一个平台使用，可以通过财付通横跨多个平台。对于趋利性很强的用户来说，虽然不能立刻得到现金返利，但结合财付通支付抵扣，还是有吸引力的。

个人认为，彩贝是通过从商家那里拿到的 CPS 和商家扣点提成，通过积分的方式从财付通这个渠道将其中一部分补贴用户，目的并不真正是返利，而是通过返利这种方式吸引用户，通过长期的习惯培养，从而形成忠诚度较高的用户群，最终会让 B2C 的大部分订单是通过 QQ 用户产生的，而后做很多其他的事情。比如通过数据挖掘，对用户进行分类画像，然后再将商品通过腾讯的渠道（tips）告知用户，未来或将提供广告投放平台给商家，甚至推广腾讯自己的 B2C。

从行业现状来看，拍拍网、QQ 商城、易迅一直被淘宝、天猫和京东打压，其他垂直电商和导购网站也是来势汹汹。所以，直观看来，彩贝积分的影响力一直不算太大。与其他返利网站相比，彩贝的优势之一是能利用 QQ 账号体系，但将来如何应对新浪微博的挑战也还是一个未知数。

值得一提的是，彩贝没有满足返利网站用户最直接的返现需求、缺少比价功能、没有商品搜索功能，虽然没有帮助用户以更低的方式找到他们喜欢的东西，但在这点上，商家和用户天生就是矛盾的，我觉得首要关注的还是商家卖家的利益。而且彩贝目前只做 B2C，可预见的未来也不可能把流量引到淘宝。这也是彩贝的一个重要的差异化。毕竟电子商务是重商务的。

机会和前景

返利积分其实是一个拼资源和拼背景的生意。对商家来说，返利网站其实只是营销渠道和工具，流量才是重要的，只有自身拥有了足够多的用户和强大的品牌认知，才能吸引商家参与进来。从我的体验感受来看，彩贝不缺少用户，但品牌宣传和营销上做得貌似还不够。

最近几年，招商银行做得很好。它与星巴克、DQ、万达影城、麦当劳、肯德基等线下知名的商家合作，全额兑换不会破坏这些商家的价格体系，让信用卡积分使用范围更广，能在线下消费掉，信用卡的品牌认知度也能借势提升，更有价值，用户也比较喜欢。对招行、商家、用户都有价值。

经过这几年的观察，我认为有两个方向还是返利网站的机会：

1. 线下商家。

我们的产品努力多年，但限于用户数量和品牌认知度，在与线下商家合作上一直困难重重，而这两点恰好是彩贝的优势（如果能结合微信那就更好了），更重要的是，这样能让积分在线上和线下流转起来，多一个应用场景，对用户也是有益，但这需要一定的 O2O 资源。

2. APP 市场

用彩贝积分吸引用户使用应用宝下载 APP，并与游戏厂商合作，内购时用彩贝积分付费。这样不仅方便用户，而且还能进一步让彩贝积分“通用”起来，另外也能提升应用宝的装机量和使用量。

产品体验

从问卷星的这份调查（ http://www.sojump.com/report/2768031.aspx ）来看，返利用户有以下几个特点：
1. 来源是通过朋友介绍和微博；
2. 收入普遍在 5000 元以下；
3. 看重的是返利比例和到账时间；
4. 习惯于找到商品后到返利网站搜索；
5. 不经常使用的原因是忘了；
6. 一旦在使用一个返利网站的同时就不会再使用其他网站了。

结合上面这份不知道是否靠谱的数据和用户使用场景，在体验过程中发现了一些彩贝可以优化的地方：

  • 网站没有向用户强调用 QQ 登录减少账号管理麻烦，完成购物还有彩贝通用积分这一优势和卖点；
  • 对新用户不友好，帮助入口太不明显了，对于积分到账时间、图标点亮方式、兑换使用需财付通等重要信息居然是 soso 百科告诉我的；
  • 虽然京东在彩贝的返现比例最高达到 6.4%，但网站上并没有表现出这一优势；
  • 虽然能用 QQ 号登录，而且也填写了个人资料，但还是需要在每个B2C重新录入电话和收货地址；
  • 我用彩贝在京东买了一个路由器，在彩贝网站上除了提醒我待返积分有多少之外，就没有任何引导了，其实可以引导用户将购物信息分享到社交网站并赢取返利，或者提醒用户将彩贝加入浏览器收藏夹；
  • 用户缺少经常打开彩贝的动机，比如没有每日签到领彩贝神马的；
  • 彩贝没有很好的运用用户的关系链，没有邀请功能，而且没有社区氛围。我尝试打开彩贝论坛，但无法登录，貌似有 bug ，论坛可以运营晒单、购物攻略、今日推荐商品、产品服务建议等内容，对用户、彩贝和商家都有帮助；
  • 运营是上没有结合当前热点，缺少人情味。除了 banenr 外，其他地方都没有情人节和元宵节相关内容了；
  • 财付通开通会员时，没有告知用户为什么开通，开通门槛又太高，估计这个步骤的流失率很高；
  • 新用户进来后，如果暂时没有购物需求，长期面对 0 积分很寂寞的啊，可以用新手任务模式先少量发放一些积分给用户，让用户账户上有一点东西让他牵挂着。如果长期没有使用彩贝，在后续营销上也有一个理由。
  • 对于趋利性很强的用户，B2C 优惠券也是一个吸引，淘宝上有很多各个商家的优惠券在售卖。彩贝如果可以得到合作商家的优惠券，那么既可以发放给用户，有可以通过积分售卖给用户，降低彩贝运营成本，也有益于商家和用户，但彩贝的优惠券类目是空的。
